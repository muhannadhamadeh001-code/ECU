# System Overview

This board is divided into functional blocks:

1. Power Input and Protection
2. Buck Converter (12V → 5V)
3. Voltage Regulation (5V → 3.3V)
4. ESP32 Microcontroller
5. CAN Bus Interface
6. USB-C and USB-UART Interface
7. User Interface and Control Signals

Each block is physically and electrically arranged to minimize noise,
ensure stable power delivery, and improve reliability.
